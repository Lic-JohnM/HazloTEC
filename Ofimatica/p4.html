Caracteres especiales y emojis en Word

Además de las letras y números del teclado, existen muchos caracteres especiales que se pueden usar para escribir símbolos, signos matemáticos, flechas, monedas, letras acentuadas o figuras decorativas.

¿Cómo se escriben los caracteres especiales?

  • Algunos se escriben con combinaciones de teclas, por ejemplo:
    • Alt + 64 → @
    • Alt + 35 → #
    • Alt + 21 → §
    • Alt + 3 → ♥
    (Para usarlos, mantén presionada la tecla Alt y escribe el número en el teclado numérico.)
  • También puedes ir a Insertar → Símbolo → Más símbolos para ver un panel con muchos más.

Emojis 😀 😎 💡

Los emojis se usan para expresar emociones o resaltar partes del texto. En Windows puedes abrir el panel de emojis con:

Presiona Windows + . (punto) o Windows + ; para abrir el panel de emojis y símbolos.

Desde ahí puedes insertar caritas 😀, animales 🐱, objetos ✏️, banderas 🚩 o símbolos matemáticos ➕. ¡Úsalos para darle vida a tus documentos, títulos o tablas!

Insertar fórmulas y símbolos

En la pestaña Insertar → Ecuación puedes crear fórmulas matemáticas como:

E = mc²      √(a² + b²) = c

También puedes usar Insertar → Símbolo para añadir signos como: © ® ™ ∞ ≤ ≥ ÷ × ° ± Σ Ω → ← ↑ ↓

Ejercicio práctico

  1. Escribe tu nombre con un símbolo decorativo al inicio y otro al final.
  2. Inserta tres emojis que representen tu estado de ánimo hoy.
  3. Agrega una ecuación matemática sencilla (por ejemplo, el área de un triángulo).
  4. Usa subrayado o color para resaltar los símbolos más interesantes.
