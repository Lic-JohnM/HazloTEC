📐 Análisis y vistas técnicas de modelos STL con Python

Este módulo enseña cómo usar Python para apoyar el dibujo técnico desde archivos STL. Aprenderás a:

  • Importar modelos STL
  • Calcular volumen, área y dimensiones
  • Exportar un reporte en Excel
  • Generar vistas (frontal, superior, lateral, isométrica)
  • Visualizar cotas básicas en las vistas

📚 Instalación de bibliotecas

pip install trimesh pandas openpyxl matplotlib

🔍 Código completo con vistas y cotas

import trimesh
import pandas as pd
import matplotlib.pyplot as plt

# Cargar modelo STL
malla = trimesh.load('modelo.stl')

# Extraer propiedades
volumen = malla.volume
area = malla.area
dimensiones = malla.extents

# Crear DataFrame y exportar a Excel
df = pd.DataFrame({
    'Volumen (mm³)': [volumen],
    'Área superficial (mm²)': [area],
    'Largo (X)': [dimensiones[0]],
    'Ancho (Y)': [dimensiones[1]],
    'Alto (Z)': [dimensiones[2]]
})
df.to_excel('reporte_modelo.xlsx', index=False)

# Crear vistas
def guardar_vista(malla, direccion, nombre_archivo):
    escena = malla.scene()
    escena.set_camera(angles=direccion)
    imagen = escena.save_image(resolution=(800, 600), visible=True)
    with open(nombre_archivo, 'wb') as f:
        f.write(imagen)

guardar_vista(malla, (0, 0, 0), 'vista_frontal.png')
guardar_vista(malla, (0, -90, 0), 'vista_lateral.png')
guardar_vista(malla, (90, 0, 0), 'vista_superior.png')
guardar_vista(malla, (45, 45, 45), 'vista_isometrica.png')

# Generar cotas básicas (dimensiones en gráfico)
plt.figure()
plt.title('Cotas del modelo (vista simplificada)')
plt.text(0.5, 0.8, f"Largo X: {dimensiones[0]:.1f} mm", ha='center')
plt.text(0.5, 0.6, f"Ancho Y: {dimensiones[1]:.1f} mm", ha='center')
plt.text(0.5, 0.4, f"Alto Z: {dimensiones[2]:.1f} mm", ha='center')
plt.axis('off')
plt.savefig('cotas.png')

📊 Resultado

El script generará un archivo Excel y las siguientes imágenes:

  • vista_frontal.png
  • vista_lateral.png
  • vista_superior.png
  • vista_isometrica.png
  • cotas.png (dimensiones rotuladas)
