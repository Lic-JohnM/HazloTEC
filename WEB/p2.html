Explorando más etiquetas HTML

Ahora que conoces las etiquetas básicas de texto, es momento de explorar otras que permiten organizar y enlazar mejor la información.

Listas

Las listas pueden ser ordenadas (<ol>) o desordenadas (<ul>).


<ul>
  <li>Manzana</li>
  <li>Banano</li>
</ul>

<ol>
  <li>Paso 1</li>
  <li>Paso 2</li>
</ol>

Imágenes

Se insertan con la etiqueta <img> usando el atributo src.

<img src="https://via.placeholder.com/150" alt="Ejemplo">

Enlaces

Para enlazar otra página, usamos <a href="URL">

<a href="https://google.com">Ir a Google</a>

Tablas

Permiten organizar datos en filas y columnas.


<table border="1">
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>12</td>
  </tr>
</table>
