Estructura de una página HTML

Una página web debe tener una estructura mínima para funcionar correctamente:


<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Mi Página</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
  </body>
</html>

¿Qué es el <head>?

Es la parte oculta de la página. Aquí se define el título, el tipo de letra, íconos, estilos, etc.

¿Qué es el <body>?

Es la parte visible de la página. Todo lo que el usuario ve en pantalla va dentro del cuerpo.
