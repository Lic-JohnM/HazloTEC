Unidades de medida en CSS

Cuando diseñamos una página web, usamos unidades para definir el tamaño de los elementos (como textos, cuadros o márgenes). Las más comunes son:

  • px (píxeles): Es la unidad más usada. Un píxel es como un punto pequeñito en la pantalla. Su tamaño real puede cambiar según el aparato (celular, computador o televisor), pero en el diseño web usamos px como si fueran medidas estándar.

  • % (porcentaje): Representa un valor relativo al tamaño de su contenedor. Por ejemplo, si un cuadro mide el 50%, ocupará la mitad del espacio donde está.

A partir de ahora, usaremos px para la mayoría de nuestros ejercicios.

Colores digitales

Los colores en CSS se pueden definir de diferentes formas:

  • Por nombre: red, blue, green
  • Por código hexadecimal: #ff0000
  • Por valores RGB: rgb(255,0,0)

Modelo de caja (Box Model)

Todo elemento en HTML tiene una "caja" con 4 partes:

  • Content: el contenido (texto, imagen, etc.)
  • Padding: espacio interno
  • Border: borde que rodea el contenido
  • Margin: espacio hacia afuera
