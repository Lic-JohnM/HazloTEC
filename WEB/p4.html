¿Qué es CSS?

CSS significa Cascading Style Sheets. Es el lenguaje que se usa para aplicar estilos a una página web, como colores, tamaños, fuentes, márgenes, etc.

1. Estilos con el atributo style

Se puede usar el atributo style dentro de una etiqueta HTML para cambiar su apariencia.


<p style="color:blue;">Este texto es azul</p>

Ejemplo en vivo: Este texto es azul

2. Estilos con la etiqueta <style>

También se puede insertar CSS dentro de la página, en la parte del <head>.


<style>
  h1 {
    color: red;
    font-family: Arial;
  }
</style>

Resultado: el título será rojo y con letra Arial.

3. Estilos comunes

  • color: color del texto
  • background-color: color de fondo
  • font-size: tamaño del texto
  • font-family: tipo de letra
  • text-align: alineación (left, center, right)
