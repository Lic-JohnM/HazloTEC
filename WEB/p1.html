Diseño Web: Introducción

El diseño web es el proceso de crear y estructurar sitios en Internet. Involucra el uso de herramientas y lenguajes que permiten mostrar información de manera atractiva, clara y funcional para los usuarios.

¿Qué es HTML?

HTML significa HyperText Markup Language. Es el lenguaje que se usa para definir la estructura y el contenido de una página web. Con HTML podemos crear títulos, párrafos, enlaces, imágenes y más.

¿Qué es una etiqueta?

Una etiqueta es una instrucción que le dice al navegador cómo debe mostrar un contenido. Todas las etiquetas HTML se escriben entre signos de menor y mayor: 

<etiqueta>Contenido</etiqueta>

Etiquetas básicas de texto

  • <b> para negrita
  • <i> para cursiva
  • <u> para subrayado
  • <br> para un salto de línea (no tiene cierre)
  • <p> para un párrafo
  • <h1> hasta <h6> para títulos (de mayor a menor tamaño)

Ejemplo de uso combinado:


<h1>Hola Mundo</h1>
<p>Este es un <b>text</b> en <i>HTML</i></p>

¿Cómo guardar tu código HTML en el Bloc de notas?

+

Para ver tu primera página web, sigue estos pasos:

  1. Abre el Bloc de notas.
  2. Escribe el siguiente código:
    3. 
<html>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Esta es mi primera página web</p>
  </body>
</html>
  3. Ve a Archivo → Guardar como...
  4. En el nombre escribe: "mi_pagina.html" (con comillas dobles)
  5. En tipo selecciona: Todos los archivos
  6. Haz clic en Guardar
  7. Haz doble clic en el archivo para abrirlo en el navegador

⚠ Importante: Usa comillas dobles para que el archivo se guarde con la extensión correcta (.html) y no como un archivo de texto común.
