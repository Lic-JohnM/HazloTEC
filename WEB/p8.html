¿Qué es JavaScript?

JavaScript es un lenguaje que se usa para agregar interactividad a las páginas web. Permite que el sitio "responda" cuando el usuario hace clic, escribe o realiza alguna acción.

¿Cómo se escribe JavaScript?

Se puede escribir directamente en HTML usando la etiqueta <script>, o con atributos como onclick.
