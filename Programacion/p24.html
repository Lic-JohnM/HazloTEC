Pygame

¿Qué es Pygame? Pygame es una biblioteca de Python diseñada para crear videojuegos. Te ofrece herramientas para manejar gráficos, sonidos y eventos del teclado o ratón, facilitando la creación de juegos y aplicaciones multimedia.

Con Pygame puedes aprender a programar juegos simples y entender conceptos como animación, detección de eventos y gestión de recursos multimedia.

Ejemplo básico de Pygame


import pygame
import sys

pygame.init()

# Configurar la ventana
screen = pygame.display.set_mode((400, 300))
pygame.display.set_caption('Hola Pygame')

# Bucle principal
while True:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            pygame.quit()
            sys.exit()
    
    screen.fill((255, 255, 255))  # Fondo blanco
    pygame.display.flip()

Este código crea una ventana de 400x300 píxeles con fondo blanco y un título. El bucle principal mantiene la ventana abierta hasta que el usuario la cierre.

Para usar Pygame, primero instala la biblioteca con:


pip install pygame

Luego guarda el código en un archivo, por ejemplo juego.py, y ejecútalo con python juego.py.

¡Así puedes empezar a crear tus propios videojuegos con Python!
