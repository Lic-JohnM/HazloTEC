Listas con Variables


En Python, las listas no solo pueden contener valores fijos, sino también variables. Esto es muy útil cuando los elementos de la lista provienen de cálculos, entradas del usuario o cualquier otra fuente dinámica. Al utilizar variables, puedes crear listas que se adapten a diferentes situaciones o condiciones durante la ejecución del programa.

Ejemplo Básico: Crear una lista con variables

Supongamos que tienes tres variables que almacenan nombres de frutas. Puedes crear una lista utilizando estas variables como elementos de la lista. Esto es especialmente útil cuando los datos de la lista se generan de manera dinámica, por ejemplo, a partir de entradas del usuario o resultados de una operación.

# Variables
elemento1 = "manzana"
elemento2 = "banana"
elemento3 = "cereza"

# Crear lista con variables
frutas = [elemento1, elemento2, elemento3]

# Imprimir la lista
print(frutas)  # Imprime: ['manzana', 'banana', 'cereza']

Usar variables para modificar listas dinámicamente

Las listas con variables también permiten modificar sus elementos de manera dinámica. Puedes realizar cambios en las variables y reflejar esos cambios automáticamente en la lista.

# Modificando los valores de las variables
elemento1 = "kiwi"
elemento2 = "mango"

# Crear nueva lista con los valores modificados
frutas = [elemento1, elemento2, elemento3]

# Imprimir la lista actualizada
print(frutas)  # Imprime: ['kiwi', 'mango', 'cereza']

Listas con entradas del usuario

También puedes usar variables que almacenan datos introducidos por el usuario para crear listas dinámicas.

# Solicitar al usuario que ingrese frutas
elemento1 = input("Ingresa una fruta: ")
elemento2 = input("Ingresa otra fruta: ")
elemento3 = input("Ingresa una fruta más: ")

# Crear lista con las frutas proporcionadas por el usuario
frutas = [elemento1, elemento2, elemento3]

# Mostrar la lista de frutas
print("Las frutas que ingresaste son:", frutas)

Operaciones con listas que contienen variables

Al tener una lista que contiene variables, puedes realizar diversas operaciones como agregar, eliminar o modificar elementos, así como iterar sobre la lista para acceder a cada elemento.

# Variables
nombre1 = "Carlos"
nombre2 = "Ana"
nombre3 = "Luis"

# Crear lista de nombres
nombres = [nombre1, nombre2, nombre3]

# Agregar un nuevo nombre
nombres.append("Sofia")

# Eliminar un nombre
nombres.remove("Ana")

# Imprimir la lista de nombres
print(nombres)  # Imprime: ['Carlos', 'Luis', 'Sofia']

Conclusión

Las listas con variables te permiten almacenar y manipular datos dinámicos de manera flexible en Python. Puedes usar variables para:

  • Crear listas basadas en cálculos.
  • Modificar elementos de la lista.
  • Recoger datos de fuentes externas, como entradas del usuario.
  • Realizar operaciones sobre listas como agregar, eliminar y modificar elementos.
