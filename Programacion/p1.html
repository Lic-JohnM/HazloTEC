Algoritmos

Un algoritmo es una secuencia de pasos que seguimos para resolver un problema o realizar una tarea. Es como una receta de cocina: si sigues los pasos correctamente, obtienes el resultado que esperas. Incluso cuando usamos un televisor, seguimos un algoritmo, aunque no nos demos cuenta. Mira este ejemplo:

  • Buscar el control remoto.
  • Prender el televisor.
  • Buscar tu canal o programa favorito.
  • Ver el programa.
  • Apagar el televisor.
  • Guardar el control remoto.

Los algoritmos nos ayudan a hacer las cosas de manera ordenada. Si usas una plataforma de streaming en lugar de la televisión, algunos pasos cambian, como abrir la aplicación y buscar una película. Pero sigue siendo un algoritmo.
