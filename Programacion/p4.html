Print


Ahora que conoces algunas de las características de Python, es importante entender cómo interactuar con el programa. Dos funciones básicas y fundamentales en Python son `print` e `input`. La función `print` se utiliza para mostrar información en la pantalla. Por ejemplo:


print("¡Hola, mundo!")
print("Python es divertido.")
print("Este es un ejemplo de print.")

Estos ejemplos muestran cómo `print` puede usarse para mostrar diferentes tipos de mensajes. Puedes experimentar con `print` en Google Colab, una herramienta en línea donde puedes ejecutar código Python directamente desde tu navegador. Haz clic en el siguiente enlace para probarlo: Google Colab.


Tipos de Comentarios


Comentarios de Línea: Se usan para explicar una sola línea o una sección breve del código. Se denotan con # en Python.

Comentarios de Bloque: Se utilizan para explicar secciones más extensas del código y se pueden abarcar múltiples líneas. Se denotan con """ en Python.


Ejemplo

A continuación, se muestra un ejemplo en Python que ilustra ambos tipos de comentarios:



# Este es un comentario de línea que explica la siguiente línea de código
print("Hola, mundo!")  # Imprime un mensaje en la consola

"""
Este es un comentario de bloque.
A continuación, realizamos una operación básica de suma
y mostramos el resultado en la consola.
"""
a = 5
b = 3
resultado = a + b
print("La suma de", a, "y", b, "es", resultado)

Los comentarios son una herramienta poderosa para hacer que el código sea más legible y fácil de mantener, y su uso adecuado es una práctica recomendada en el desarrollo de software.
