Figuras 2D en OpenSCAD


OpenSCAD permite crear muchas figuras planas (2D) usando instrucciones simples. Estas figuras se pueden combinar, modificar o convertir en 3D más adelante. Aquí te mostramos las más comunes:


1. Cuadrado o Rectángulo - square()

Se usa para crear cuadrados o rectángulos. Puedes ajustar el tamaño con [ancho, alto].

square([40, 20]);

2. Círculo - circle()

Crea un círculo indicando su radio. Entre más alto el $fn, más suave será el borde.

circle(r = 20, $fn = 50);

3. Texto - text()

También se puede escribir texto plano que luego puede extruirse en 3D.

text("Hola", size=20);

4. Polígono libre - polygon()

Permite crear cualquier forma conectando puntos. Ya lo has usado, pero recuerda que es útil para figuras irregulares:

polygon(points = [[0,0],[30,0],[15,25]]);

Estas figuras se pueden combinar fácilmente, por ejemplo:


square([30, 30]);
translate([40, 0]) circle(r = 15);
translate([80, 5]) text("Hola", size = 10);
