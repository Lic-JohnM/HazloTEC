🔁 Módulos y Reutilización de Código

Los módulos en OpenSCAD permiten crear piezas reutilizables. Esto es útil para no repetir código cuando diseñamos muchas figuras similares, como fichas, patas de mesa, botones o estructuras repetidas.

🧩 ¿Cómo crear un módulo?

Un módulo es como una función. Lo defines una vez y lo puedes usar muchas veces: 

module ficha() {
    cylinder(h=5, r=10);
}

Luego puedes usarlo así: 

ficha();
translate([25,0,0]) ficha();
translate([50,0,0]) ficha();

🧪 Ejemplo: 5 fichas alineadas

module ficha(colorFicha="red") {
    color(colorFicha)
        cylinder(h=5, r=10);
}

for (i = [0:4]) {
    translate([i*25, 0, 0])
        ficha("blue");
}

Este ejemplo crea 5 fichas azules reutilizando el mismo módulo.

📦 Ventajas de usar módulos

  • Evitas repetir código
  • Puedes modificar todo desde un solo lugar
  • Organizas mejor tus diseños grandes
  • Permiten usar parámetros personalizados

💡 Tip:

Un módulo puede recibir parámetros como altura, color o forma para ser aún más flexible.
