Operaciones de Corte en OpenSCAD


OpenSCAD permite combinar figuras con operaciones llamadas booleanas: unir, cortar o encontrar lo que tienen en común.

1. Restar una figura con difference()

Sirve para cortar una figura usando otra. Aquí vemos cómo hacer un rectángulo con una esquina curva:


difference() {
    square([40, 40]);        // figura base
    translate([40, 40])
        circle(r = 20);      // esquina curva que se "corta"
}

2. Unir figuras con union()

Combina dos o más figuras para que actúen como una sola:


union() {
    square([30, 30]);
    translate([30, 0]) circle(r = 15);
}

3. Intersección con intersection()

Deja solo la parte donde dos figuras se superponen:


intersection() {
    square([40, 40]);
    translate([20, 20]) circle(r = 30);
}

Estas herramientas son muy poderosas para dar forma precisa a tus diseños. Con un poco de práctica, puedes hacer recortes complejos, curvas suaves o unir piezas personalizadas.
