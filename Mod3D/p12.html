🐉 Introducción a Blender

Blender es un programa libre y gratuito para crear gráficos y animaciones 3D. Se usa en películas, videojuegos, ilustración digital y más. Aunque tiene muchas herramientas avanzadas, también puedes usarlo para crear modelos sencillos desde el primer día.

🛠️ ¿Qué se puede hacer en Blender?

  • Modelado 3D artístico
  • Escultura digital
  • Animación de personajes y objetos
  • Renderizado con luces y materiales
  • Edición de video, simulaciones, efectos visuales ¡y más!

🎮 Interfaz para principiantes

Al abrir Blender, verás un cubo. Puedes moverlo, escalarlo o rotarlo. También puedes agregar nuevas figuras desde el menú Add → Mesh (como esferas, conos o monos).

🧪 Actividad: Primer contacto con Blender

  1. Abre Blender (descárgalo desde blender.org)
  2. Mueve el cubo usando la tecla G (grab)
  3. Escálalo con S o rótarlo con R
  4. Agrega una figura nueva (por ejemplo, un mono: Shift + A → Mesh → Monkey)
  5. Intenta cambiar el color en el panel Materiales

🎨 ¿Qué diferencia tiene con OpenSCAD?

Blender se enfoca más en el arte y la expresión visual, mientras que OpenSCAD se basa en código y precisión matemática. Ambos son útiles, pero tienen propósitos diferentes.

💡 Tip:

Puedes usar Blender como complemento: para darle forma y color a ideas creativas, y OpenSCAD para piezas que necesiten medidas exactas.
