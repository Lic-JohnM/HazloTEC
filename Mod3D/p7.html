🔄 Extrusión en OpenSCAD

La extrusión permite transformar figuras 2D en modelos 3D. Existen dos tipos principales:

  • linear_extrude(): extruye una figura en línea recta.
  • rotate_extrude(): gira una figura 2D alrededor del eje Y para crear sólidos de revolución.

🧱 Extrusión lineal básica

Convierte una figura 2D en un volumen extruyéndola hacia arriba: 

linear_extrude(height=20)
    text("Hola", size=10);

🌀 Extrusión con giro (no lineal)

rotate_extrude() genera figuras girando un perfil 2D alrededor del eje Y, ideal para vasos, platos o jarrones: 

rotate_extrude()
    translate([10, 0])
        circle(5);

Este ejemplo crea un toro (rosquilla) al girar un círculo.

🧪 Ejemplo: jarrón con perfil

rotate_extrude()
    polygon(points=[[0,0], [5,0], [7,10], [3,20], [0,25]]);

Gira un perfil para formar un jarrón de revolución.

🌀 Extrusión con torsión (twist)

También puedes torcer la figura a medida que se extruye. Esto da formas helicoidales: 

linear_extrude(height=40, twist=180, slices=100)
    circle(5);

Este ejemplo crea una columna torcida (tipo tornillo).

📏 Extrusión con escala

Puedes hacer que la forma cambie de tamaño mientras se extruye: 

linear_extrude(height=20, scale=0.5)
    square([20, 20], center=true);

Esto crea una pirámide truncada a partir de un cuadrado.

💡 Recomendaciones

  • Usa $fn para controlar la suavidad de curvas.
  • Siempre traduce tu figura si necesitas alejarla del eje Y para usar rotate_extrude().
  • El parámetro slices mejora la precisión de formas torcidas.
