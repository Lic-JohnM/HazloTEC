🧠 Variables y Bloques Personalizados

Las variables permiten que un juego recuerde cosas como el puntaje, las vidas, los niveles o el tiempo. Son como cajas donde se guarda información que puede cambiar durante el juego.

Los bloques personalizados sirven para agrupar instrucciones y reutilizarlas, como si crearas tus propios comandos. Esto hace que el código sea más limpio y fácil de entender.

🎮 Ejemplo: Juego de aventuras tipo Zelda

En los juegos de aventuras, necesitas guardar muchas cosas: cuántas llaves tienes, si encontraste un tesoro, o si tu energía es baja. Las variables y bloques te ayudan a manejar todo esto.

🔧 Actividad práctica en Scratch

Vamos a crear un juego donde el personaje tiene que recoger llaves y escapar por una puerta secreta:

  1. Crea una variable llamada "Llaves"
  2. Cuando el personaje toque una llave, suma 1 a la variable y oculta el objeto
  3. Cuando llegue a la puerta, verifica si "Llaves = 3" para dejarlo pasar
  4. Crea un bloque personalizado que revise si el personaje tiene todas las llaves

💡 Consejo

Con variables también puedes controlar la dificultad, crear niveles, manejar tiendas o desbloquear personajes. Y con bloques personalizados, puedes repetir acciones como "dañar al enemigo" o "mostrar mensaje" sin volver a escribir todo.
